ModeloLogicoEISRUN 0..* Base Modelo Logico para el tipo de dato RUN
Instances of this logical model can be the target of a Reference
run Σ 0..1 string El RUN es el n° de identificación que entrega la mayor credibilidad para la individualización inequívoca de las personas y se debe utilizar en todas las personas que lo poseen. En Chile es único, irrepetible e intransferible que posee todo chileno resida o no en Chile.
digitoVerificador Σ 0..1 string Parte de la estructura del RUN y debe permitir valor numérico, incluído el cero y la letra k mayúscula o minúscula.
